# Snakemake Integration (Seqnado)

Integration with Snakemake and the `seqnado` directory structure.

This example shows how to:
- Pass parameters from Snakemake to TrackNado.
- Use the `from_seqnado_path` extractor.
- Automate hub staging as part of a workflow.
