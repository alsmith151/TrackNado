# CapCruncher Hub Generation

This example replicates the complex logic of a legacy pipeline for capturing-based 3C data (CapCruncher).

It demonstrates:
- Custom metadata extraction from specific directory structures.
- Complex grouping of BigWig signals.
- Custom track coloring.
